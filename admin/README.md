# Runic Text / Svensk Runtextdatabas

## Quick start
Install docker from https://www.docker.com/community-edition.

In the root directory of the cloned repo, run `docker-compose up`.

Go to http://locahost:8080 to access the adminstration application.

Go to http://localhost:3030 to access the Fuseki triple store.

Updates to the code in the `src` directory should be immediately visible in the administration application.
